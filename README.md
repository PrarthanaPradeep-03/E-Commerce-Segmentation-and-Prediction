# E-Commerce Customer Segmentation and Prediction

## Project Overview
This project analyzes e-commerce customer data to segment users based on purchasing behavior and predict future purchase patterns. It helps businesses understand customers and make data-driven decisions.

## Objective
- Identify customer segments
- Analyze buying behavior
- Predict future purchases
- Improve retention and marketing strategies

## Tools & Technologies
- Python
- Pandas, NumPy
- Scikit-learn
- Matplotlib, Seaborn

## Dataset
E-commerce transactional dataset containing customer purchase history, frequency, and spending details.

## Project Workflow
1. Data Cleaning & Preprocessing
   - Handled missing values and duplicates
   - Converted data types

2. Exploratory Data Analysis (EDA)
   - Analyzed patterns and trends
   - Visualized data distributions

3. RFM Analysis
   - Recency: Last purchase
   - Frequency: Number of purchases
   - Monetary: Total spending

4. Customer Segmentation
   - Applied K-Means clustering
   - Grouped customers into segments

5. Prediction Model
   - Built ML models to predict behavior
   - Evaluated using accuracy, precision, recall, F1-score

## Key Insights
- Identified high-value and low-value customers
- Detected frequent and inactive users
- Enabled targeted marketing strategies
